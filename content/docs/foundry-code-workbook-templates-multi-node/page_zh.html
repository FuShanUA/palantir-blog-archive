多节点模板

Code Workbook 支持通过多节点模板来实现工作流的模板化。你可以基于多个现有模板来创建新模板，并将这些模板中的参数值绑定在一起。

下面我们以 titanic_dataset 为例进行说明。我们将把一个工作流模板化：选择两个数值列，并将它们绘制在两张图表中。第一张图是这两个数值列的散点图。第二张图是在对数值列进行筛选的基础上，绘制乘客是否幸存的直方图。

以下是我们正在模板化的工作流的可视化概览：

创建模板

首先，创建一个模板，从输入数据集中选择两个数值列以及 Survived 列。将该模板命名为 Filter。

其次，创建一个绘制这两个数值列散点图的模板。将该模板命名为 Scatterplot。请注意，输入数据集被设置为以 Pandas 数据帧的形式读取。

最后，创建一个模板，基于这两个数值列和两个模板化输入对输入数据集进行筛选。请注意，输入数据集被设置为以 Spark 数据帧的形式读取。

创建多节点模板

选中全部三个模板并右键打开菜单，然后选择 Create new template。此时你应该能看到模板编辑器。

我们希望将 col1 参数的值关联起来，同时将 col2 参数的值也关联起来。首先，点击进入 Filter 模板。点击 col1，在下拉菜单中选择 Create shared parameter。

在右侧面板中，会创建一个名为 col1 的新参数。在右侧面板中选择 titanic_dataset 作为源数据集。接着，点击进入另外两个模板，并选择将它们的 col1 与新建的 col1 多节点模板参数进行关联。对 col2 重复上述操作，然后保存该多节点模板。

使用多节点模板

我们之前创建的三个模板现在已成为一个多节点模板的一部分。

点击进入 Filter 模板。在 col1 和 col2 参数旁边，会有一个图标指示该参数值由多节点模板参数控制。

在面板顶部选择 View Group。此时你会看到一个视图，它高亮显示了多节点模板中的节点，并列出了模板中的共享参数。你可以在此视图中更改 col1 和 col2 的值，三个节点中的所有实例也会随之更改。

如果在子节点面板中更改共享参数的值，多节点模板中所有实例的该参数值也会同步更改。
