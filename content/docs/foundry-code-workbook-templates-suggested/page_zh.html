建议模板

对于一组相似的数据集，您可能需要建议要应用的模板。例如，对于一组包含月度股票回报的数据集，您可能希望推荐一个用于计算年化回报的模板。建议模板会显示在“新建转换”面板的一个独立按钮中，从而提高其可发现性。

在模板中配置输入标签

下文展示了一个根据月度回报计算年化回报的模板。对于任何带有 Finance 标签的数据集，当用户以该数据集为父级新建转换时，此模板将出现在 Suggested（建议）选项卡中。

在创建或编辑模板时，于 Input Tags（输入标签）字段中配置所需的输入标签。您可以添加多个输入标签。

向数据集添加标签

向数据集添加输入标签的一种方法是在 Dataset-app 中打开该数据集。请注意，由于标签与数据集绑定，因此无法在 Code Workbook 中为未持久化的转换添加标签。这意味着未持久化的转换不支持建议模板功能。

在多节点模板中配置输出标签

> 转换：无需从 Dataset-app 手动添加标签，您可以直接在多节点模板中配置输出标签。 > 假设有一个包含两个节点的多节点模板，其中第二个节点的输出是一个包含月度股票回报的数据集。我们希望该数据集自动添加 Finance 标签，这样配置了该输入标签的模板就会出现在“建议模板”中。 > 创建或编辑多节点模板时，请在 Output tags（输出标签）下方设置所需的输出标签。应用该多节点模板后，输出标签将立即添加到由该多节点模板生成的数据集中。
