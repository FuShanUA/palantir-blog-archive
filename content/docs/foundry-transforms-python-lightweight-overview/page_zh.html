Python 转换

Python 是在 Foundry 中编写数据转换功能最全面的语言。它支持批处理和增量管道，支持创建和共享代码库以实现复用，并允许在代码中定义数据期望，从而确保数据管道的高质量。Python 转换可以配置为单节点或多节点计算引擎。
