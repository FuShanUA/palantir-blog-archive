创建图表

由于 titanic_dataset 包含行（代表乘客）和列（代表乘客的相关信息），我们可以使用柱状图，针对特定的乘客属性（如性别或客舱等级）来可视化乘客数量。例如，以下是按性别对乘客进行的分类统计：

在本示例中，我们将创建一个转换来生成所需的图表。请创建一个名为 bar_chart_of_row_counts 的 Python 转换，并插入以下代码：

1def bar_chart_of_row_counts(titanic_dataset):
2    import matplotlib.pyplot as plt
3    from pyspark.sql import functions as F
4    import numpy as np
5
6    input_df = titanic_dataset
7    categorical_column = "Sex"
8
9    # 计算数量
10    total = input_df \
11        .groupBy(categorical_column) \
12        .agg(F.count("*").alias("count")) \
13        .orderBy("count")
14
15    # 将汇总数据集转换为 pandas 格式
16    total_pdf = total.toPandas()
17
18    # 绘图代码
19    fig = plt.figure()
20    ax = fig.add_subplot(111)
21
22    y_pos = np.arange(len(total_pdf[categorical_column]))
23    ax.set_yticks(y_pos)
24    ax.barh(y_pos,total_pdf["count"])
25    ax.set_yticklabels(total_pdf[categorical_column])
26    plt.xlabel("count")
27    plt.ylabel(categorical_column)
28
29    plt.tight_layout()
30    plt.show()
31
32    # 返回聚合后的数据框以将其保存为数据集
33    return total

创建模板


接下来，我们将把这个转换改造为模板，实现通用化和复用。点击代码编辑器右上角的操作按钮，然后点击创建模板。




系统会带你进入全屏编辑器的模板创建视图。
在模板编辑器中，可以修改模板名称、描述和参数。我们将该模板命名为 Bar Chart of Row Counts by Categorical Variable，并添加以下描述：Create a transform with a bar chart of the row counts of 1 categorical column in any input dataset.




所有输入数据集（此处即 titanic_dataset）都会自动添加为模板中 dataset 类型的参数。点击模板编辑器中的 titanic_dataset 即可修改。为了让模板具备通用性，我们将参数名从 titanic_dataset 改为 input_dataset，并补充描述。




转换代码中会高亮显示两处 {{{input_dataset}}}。接下来，我们对输入列进行参数化。要将代码体中的变量指定为模板的输入参数，请点击转换右上角的添加新参数，并在代码中高亮选中所选变量。按照下图所示，高亮选中字符串 "Sex"：






这会将这部分代码作为 column 类型的参数添加，并将源数据集设为 input_dataset。在模板编辑器中，编辑 param1 参数名，将其重命名为 selected_column。


在此示例代码中，我们在转换顶部将列名定义为变量，因此只需对其进行一次参数化。在对其他转换进行模板化时，你可以使用添加按钮添加同一参数的更多实例。


接下来，选择此模板是否默认保存为数据集。勾选另存为数据集复选框后，添加时该模板将默认作为持久化转换添加。如果不勾选另存为数据集，模板将默认作为非持久化转换应用。在本例中，由于我们希望在其他应用中使用输出结果，因此选择默认“另存为数据集”。




最后，点击创建模板按钮以创建并保存模板。每次创建新模板时，都需要选择一个文件夹进行保存。在本示例中，你可以将模板保存在个人主目录中。


模板只能被拥有访问权限的用户发现和使用，因此你可以在开发期间将其保存在个人主目录中，待准备推广供更广泛使用时，再将其移至共享文件夹。模板也可以添加到数据目录中。


使用模板


创建并保存模板后，你就可以通过点击界面的方式使用模板。




要查看可用模板，请在转换创建视图中点击浏览所有模板。




浏览模板功能支持按名称、描述和标签搜索模板，也支持基于收藏夹、最近使用的模板或文件结构进行浏览。要应用某个模板，请点击其名称并选择“选择”。让我们添加刚才创建的模板。


将模板添加到图中后，你可以重命名转换并填写输入项。点击点击此处添加数据集，然后点击图中的 titanic_dataset，以此分配 input_dataset 参数。现在你可以选择任意列作为 selected_column 参数，基于该列创建图表。


选择运行以计算转换。对于输出可视化结果的转换，图视图是图中的默认视图。你可以右键点击，选择编辑，然后选择显示表视图，将该节点以表格形式查看。


编辑模板


如需更新模板背后的代码，请点击操作，然后点击编辑模板，进入代码编辑器对模板进行编辑。
