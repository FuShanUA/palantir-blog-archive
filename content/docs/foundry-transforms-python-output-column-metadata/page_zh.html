输出列元数据

在代码仓库转换中，您可以读取和写入输出数据集的列描述与列类型。

在代码仓库转换中更新列描述

您可以通过向 TransformOutput 的 write_dataframe() 函数提供可选的 column_descriptions 参数，为输出数据集添加列描述。

  • 该参数应为一个 dict ↗，其键为列名，值为列描述。列描述的长度上限为 800 个字符。
  • 代码会自动计算您的 DataFrame ↗ 中可用的列名与您提供的 dict ↗ 中键的交集，因此不会尝试为不存在的列添加描述。

示例：在代码仓库转换中写入列描述


```python
from transforms.api import transform, Input, Output

@transform(
    my_output=Output("/my/output"),
    my_input=Input("/my/input"),
)
def my_compute_function(my_input, my_output):
    my_output.write_dataframe(
        my_input.dataframe(),
        column_descriptions={
            "col_1": "col 1 description"
        }
    )
```

列类型类


column_typeclasses 属性返回一个结构化的 Dict<str, List<Dict<str, str>>>，用于将列名映射到其对应的列类型类。


    

  • List 中的每个类型类都是一个 Dict[str, str] 对象。

      

    • 此 Dict 对象只能使用 "name" 和 "kind" 作为键。每个键映射到用户想要的相应字符串。
      • 

    

    • 



column_typeclasses 的一个示例值为 {"my_column": [{"name": "my_typeclass_name", "kind": "my_typeclass_kind"}]}。
