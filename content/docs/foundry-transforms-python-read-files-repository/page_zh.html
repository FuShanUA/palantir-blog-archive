读取代码库中的文件

你可以将代码库中的其他文件读取到转换上下文中。这在你设置参数供转换代码引用时非常有用。

首先，在你的 Python 代码库中编辑 setup.py：

Copied!
1setup(
2    name=os.environ['PKG_NAME'],
3# ...
4    package_data={
5        '': ['*.yaml', '*.csv']
6    }
7)

这会指示 Python 将 yaml 和 csv 文件打包到程序包中。 接着，将配置文件（例如 config.yaml，也可以是 csv 或 txt 文件）放在你的 Python 转换脚本旁边（例如下文所示的 read_yml.py）：



Copied!
1- name: tbl1
2  primaryKey:
3  - col1
4  - col2
5  update:
6  - column: col3
7    with: 'XXX'

你可以通过以下代码在你的转换脚本 read_yml.py 中读取该配置：



已复制！
1from transforms.api import transform_df, Input, Output
2from pkg_resources import resource_stream
3import yaml
4import json
5
6@transform_df(
7    Output("/Demo/read_yml")
8)
9def my_compute_function(ctx):
10    stream = resource_stream(__name__, "config.yaml")
11    docs = yaml.safe_load(stream)
12    return ctx.spark_session.createDataFrame([{'result': json.dumps(docs)}])

因此，你的项目结构如下：

  • some_folder
    • config.yaml
    • read_yml.py

这将在你的数据集中输出一行包含列“result”的数据，其内容为：



已复制！
1[{"primaryKey": ["col1", "col2"], "update": [{"column": "col3", "with": "XXX"}], "name": "tbl1"}]
