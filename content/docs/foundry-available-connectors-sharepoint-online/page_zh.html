SharePoint Online

连接至 SharePoint Online，将指定 SharePoint 库中的文件导入 Foundry。

支持的功能

功能状态
探索🟢 正式可用
批量导入🟢 正式可用
增量同步🟢 正式可用
导出任务🟡 已废弃
文件导出🟢 正式可用

数据模型

该连接器可将任意类型的文件传输至 Foundry 数据集中。文件格式将保留，传输过程中或传输完成后均不应用模式。请对输出数据集应用所需模式，或编写下游转换来访问数据。

性能与限制

可传输的文件大小没有限制。但网络问题可能导致大规模传输失败。尤其是运行时间超过两天的 Foundry 同步将被中断。为避免网络问题，建议使用较小的文件大小，并限制每次同步执行时摄入的文件数量。同步任务可定期调度运行。

不支持连接到本地 SharePoint 服务器。请使用 REST API 源类型来连接本地 SharePoint。

设置

  1. 打开数据连接应用，并在屏幕右上角选择 + 新建源。
  2. 从可用的连接器类型中选择 SharePoint Online。
  3. 按照额外的配置提示，使用以下章节中的信息继续设置连接器。

详细了解如何在 Foundry 中设置连接器。

身份验证

SharePoint Online 源的身份验证需要在 Microsoft Entra ID（前称 Azure Active Directory）中创建一个应用程序。如果您不是 Entra ID 管理员，请联系 IT 部门申请访问权限。

按照以下初始步骤获取 Azure 应用凭据：

  1. 按照微软官方文档 ↗中的说明在 Azure 中创建应用注册。
    • 在第 5 步中，选择 仅此组织目录中的帐户，并跳过 重定向 URL (可选)。
  2. 注册完成后，记录客户端 ID 和租户 ID。

接下来，在两种可用的身份验证方法之间进行选择：

  • 客户端凭据：需要对每个 SharePoint 站点进行广泛访问权限时推荐使用。
  • 用户名/密码：限制对一个或少数几个 SharePoint 站点的访问时推荐使用。

客户端凭据

在 Microsoft Entra 管理中心，完成以下步骤：

  1. 在左侧导航栏中转到 API 权限。

  2. 选择 添加权限。

  3. 选择 Microsoft Graph。

  4. 选择 应用程序权限。

    • 如果希望您的应用程序读取所有 SharePoint 站点，请添加 Sites.Read.All。
      • 如果计划配置导出任务，请改用 Sites.ReadWrite.All。
    • 如果希望您的应用程序读取选定的 SharePoint 站点，请添加 Sites.Selected。

  5. 如果您是 Entra 管理员，请选择 代表 [租户] 授予管理员同意。

  6. 如果上一步添加了 Sites.Selected，请将您的应用程序添加到特定站点 ↗。

    • "roles" 数组参数的可用选项为 "write" 和/或 "read"。使用 "read" 选项即可满足从 SharePoint 站点摄入文件的需求。
    • 为了方便地发送带有正确身份验证的 POST 请求，请使用 Graph 浏览器 ↗。
    • 您可以通过向 https://graph.microsoft.com/v1.0/sites/[tenantName]:/sites/[siteName] 发送 GET 请求来获取站点的元数据（例如：https://graph.microsoft.com/v1.0/sites/contoso.sharepoint.com:/sites/mySite）。此请求将返回一个由多个值复合而成的 ID：站点集合主机名、站点集合唯一 ID 和站点唯一 ID，其中中间的值即为执行权限 POST 请求所需的 siteId。

  7. 生成客户端密钥。 ↗

用户名/密码

用户名/密码流程涉及创建一个可以登录 Microsoft 365 的用户账户。Graph API 不支持用户名/密码认证方式的双因素身份验证。因此，我们强烈建议创建至少 32 个字符长度的随机生成密码。

在您的 Entra 管理中心，完成以下步骤：

  1. 在左侧边栏转至 API Permissions。
  2. 选择 Add a Permission。
  3. 选择 Microsoft Graph。
  4. 选择 Delegated Permissions。
  5. 添加 Sites.Read.All 权限；。
    • 如果您计划配置导出任务，请改用 Sites.ReadWrite.All。
  6. 如果您是 Azure 管理员，请选择 Grant admin consent for [tenant]。
  7. 在左侧边栏转至 Authentication。
  8. 将 Allow public client flows 更改为 Yes。
  9. 在 Microsoft Entra ID 中创建用户并使用至少 32 个字符的随机生成密码。
  10. 将该用户添加到您希望其读取或写入的任何 SharePoint 站点中。

SharePoint 加载项的基于 XML 的权限管理

如果您正在使用《SharePoint 加载项进行授权和认证 ↗》，且您的 SharePoint 加载项使用 XML 进行权限管理，您必须确保在范围 URI 中设置了正确的范围，以避免连接到 SharePoint 时出现访问问题。

请按照以下步骤验证并配置正确的范围：

  1. 找到包含您的 SharePoint 加载项权限设置的 AppManifest.xml 文件。
  2. 在 AppManifest.xml 文件中，识别 XML 文件内的范围 URI，其内容应类似于以下格式：

<AppPermissionRequests AllowAppOnlyPolicy="true"> <AppPermissionRequest Scope="http://sharepoint/content/sitecollection/web" Right="FullControl" /> </AppPermissionRequests>。

  1. 验证范围值（在此示例中为 http://sharepoint/content/sitecollection/web）是否与您正在连接的 SharePoint 站点相匹配；如果范围值不匹配，请相应地调整范围值。

网络

SharePoint Online 连接器需要通过 443 端口访问以下域名的网络权限：

  • login.microsoftonline.com
  • graph.microsoft.com
  • 您的 SharePoint URL；例如 contoso.sharepoint.com

如果您正在使用 GovCloud SharePoint 实例，请改用通过 443 端口访问以下域名：

  • login.microsoftonline.us
  • graph.microsoft.us
  • 您的 SharePoint URL；例如 contoso.sharepoint.us

接入 SharePoint 列表

SharePoint Online 连接器仅支持基于文件的接入方式。若要接入来自 SharePoint 列表的数据，请结合 Microsoft Graph API 使用外部转换。以下辅助类负责处理 OAuth2 身份验证，并提供了用于获取列表及列表项的方法，同时支持自动分页处理：


```python
import requests
import logging
from typing import Optional, Dict, List
from urllib.parse import urlparse


class SharePointListReader:
    """
    通过 Microsoft Graph API 读取 SharePoint 列表的客户端。

    该类处理 OAuth2 身份验证并提供以下方法：
    - 从 SharePoint 站点检索所有列表
    - 通过自动分页从特定列表中获取项目

    参数:
        tenant_id: Azure AD 租户 ID
        client_id: Azure AD 应用程序（客户端）ID
        client_secret: Azure AD 应用程序客户端密钥
        site_url: 完整的 SharePoint 站点 URL（例如 https://contoso.sharepoint.com/sites/mysite）
        logger: 可选的自定义日志记录器实例
    """

    def __init__(
        self,
        tenant_id: str,
        client_id: str,
        client_secret: str,
        site_url: str,
        logger: Optional[logging.Logger] = None
    ):
        self.tenant_id = tenant_id
        self.client_id = client_id
        self.client_secret = client_secret
        self.site_url = site_url.rstrip("/")
        self.base_url = "https://graph.microsoft.com/v1.0"

        self.access_token: Optional[str] = None
        self.site_id: Optional[str] = None
        self.logger = logger or self._setup_default_logger()

    def _setup_default_logger(self) -> logging.Logger:
        """配置默认的控制台输出日志记录器。"""
        logger = logging.getLogger(__name__)
        if not logger.handlers:
            handler = logging.StreamHandler()
            formatter = logging.Formatter('%(levelname)s: %(message)s')
            handler.setFormatter(formatter)
            logger.addHandler(handler)
            logger.setLevel(logging.INFO)
        return logger

    def get_access_token(self) -> bool:
        """
        从 Azure AD 获取 OAuth2 访问令牌。

        返回:
            如果成功获取令牌则返回 True，否则返回 False
        """
        token_url = f"https://login.microsoftonline.com/{self.tenant_id}/oauth2/v2.0/token"
        payload = {
            "grant_type": "client_credentials",
            "client_id": self.client_id,
            "client_secret": self.client_secret,
            "scope": "https://graph.microsoft.com/.default",
        }

        try:
            response = requests.post(token_url, data=payload)
            response.raise_for_status()
            token_data = response.json()
            self.access_token = token_data["access_token"]

            expires_in = token_data.get("expires_in", 3600)
            self.logger.info(f"Authentication successful (expires in {expires_in}s)")
            return True

        except requests.exceptions.RequestException as e:
            self.logger.error(f"Authentication failed: {e}")
            return False

    def _make_graph_request(self, url: str, params: Optional[Dict] = None) -> Optional[Dict]:
        """
        向 Microsoft Graph API 执行经过身份验证的 GET 请求。

        参数:
            url: 完整的 Graph API 端点 URL
            params: 可选的查询参数

        返回:
            以字典形式返回 JSON 响应，失败时返回 None
        """
        if not self.access_token and not self.get_access_token():
            return None

        headers = {
            "Authorization": f"Bearer {self.access_token}",
            "Content-Type": "application/json",
        }

        try:
            response = requests.get(url, headers=headers, params=params)
            response.raise_for_status()
            return response.json()

        except requests.exceptions.RequestException as e:
            self.logger.error(f"API request failed: {e}")
            if hasattr(e, 'response') and e.response is not None:
                self.logger.debug(f"Response details: {e.response.text}")
            return None

    def get_site_id(self) -> Optional[str]:
        """
        从站点 URL 检索 SharePoint 站点 ID。

        返回:
            站点 ID 字符串，如果检索失败则返回 None
        """
        if self.site_id:
            return self.site_id

        parsed = urlparse(self.site_url)
        hostname = parsed.hostname
        site_path = parsed.path.strip("/")

        url = f"{self.base_url}/sites/{hostname}:/{site_path}"
        data = self._make_graph_request(url)

        if data and "id" in data:
            self.site_id = data["id"]
            self.logger.debug(f"Site ID retrieved: {self.site_id}")
            return self.site_id

        self.logger.error("Failed to retrieve site ID")
        return None

    def get_all_lists(self) -> Optional[Dict]:
        """
        从 SharePoint 站点检索所有列表。

        返回:
            包含列表元数据的字典，失败时返回 None
        """
        site_id = self.get_site_id()
        if not site_id:
            return None

        url = f"{self.base_url}/sites/{site_id}/lists"
        data = self._make_graph_request(url)

        if data and "value" in data:
            self.logger.info(f"Found {len(data['value'])} lists in site")
            for lst in data["value"]:
                self.logger.info(f"  - {lst['name']} (ID: {lst['id']})")

        return data

    def get_all_list_items(self, list_id: str) -> Optional[List[Dict]]:
        """
        通过自动分页从 SharePoint 列表中检索所有项目。

        参数:
            list_id: SharePoint 列表的 GUID

        返回:
            项目字典列表，失败时返回 None
        """
        site_id = self.get_site_id()
        if not site_id:
            return None

        all_items = []
        url = f"{self.base_url}/sites/{site_id}/lists/{list_id}/items"
        params = {"$expand": "fields", "$top": 5000}

        page_count = 0
        while url:
            current_params = None if "@odata.nextLink" in url else params
            data = self._make_graph_request(url, current_params)

            if not data or "value" not in data:
                break

            page_count += 1
            items_in_page = len(data["value"])
            all_items.extend(data["value"])

            self.logger.debug(f"Page {page_count}: retrieved {items_in_page} items")

            url = data.get("@odata.nextLink")
            params = None

        self.logger.info(f"Retrieved {len(all_items)} total items from list")
        return all_items
```
