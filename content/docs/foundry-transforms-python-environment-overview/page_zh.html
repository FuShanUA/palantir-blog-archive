Python 环境

转换所用的 Python 环境 会在检查阶段通过 Hawk 包管理器，依据 conda_recipe/meta.yaml 文件中指定的包列表进行解析。通过包选项卡，你可以发现可用的包，并自动将其添加到 meta.yml 中以完成环境解析。解析后的环境会发布到内部的 Artifacts 供构建阶段使用。

构建转换时，系统会拉取环境文件并安装其中指定的所需包。若因故失败，转换操作会在构建期间再次使用 Hawk 重新解析环境。
