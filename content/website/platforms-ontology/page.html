Ontology

The central system for orchestrating decisions across Human+AI teams

The Ontology System encodes the data, logic, action, and security of the enterprise to automate decisions across your operations.

Model the components of
Human+AI decisions.

Encode the data of the enterprise.

Unify the vast and fragmented sources of data that provide a real-time picture of the decision space — ERP, CRM, systems of record, geospatial repositories, real-time sensors, and document stores — into coherent objects, properties, and links.

READ MORE

Capture the logic of the enterprise.

Continuously encode the reasoning and logic elements used to calculate and determine actions — spanning the complete spectrum of business rules and decision frameworks, and built to evolve as that reasoning changes.

READ MORE

Model the actions of the enterprise.

Represent real-world actions as first-class primitives and define how they can be performed — from simple transactions to sophisticated multi-step workflows that write back to operational and edge systems.

READ MORE

Govern the human-agent labor force.

Orchestrate security and governance policies across an increasingly hybrid labor force — enforcing granular controls across data, logic, and action simultaneously, whether the actor is a human or an agent.

Activate your existing infrastructure.

The Ontology securely coordinates read operations (e.g., queries and subscriptions to state changes), as well as write operations (e.g., transactional updates and high-scale batch mutations) across heterogeneous infrastructure.

Operate in sync.

ONTOLOGY TOOLCHAIN

Empower developers and agents to treat
the Ontology as a backend.

Build rich AI-enabled applications and services.

Ontology SDK (OSDK) provides a powerful developer toolchain, and complements a rich collection of DevOps capabilities that enable developer teams to deliver operational use-cases — from wildfire response, to naval logistics, to automotive assembly, among countless others.

READ MORE

Create tools for any human or any agent.

The Ontology is a "tool factory" that lets your builders define tools for both humans and agents. Tools can be created to query any data type, call any model or piece of logic, or execute any action — all governed with a rigorous and comprehensive security framework.

READ MORE

Power external agents with Ontology MCP.

Ontology MCP exposes your ontology primitives to external agents as MCP tools. External agents and systems can read objects, query data, and execute predefined actions. The same security controls that govern Palantir-native agents apply equally to external agents, enabling trusted interactions with your ontology as your agent ecosystem grows.

READ MORE

Turn specialized expertise into shared infrastructure.

Teams from across the enterprise — whether operators, data scientists, analysts, or data engineers — can compose reusable tools that encode their domain expertise directly into the Ontology. Tools power interlocking workflows and automations, accelerating development while ensuring consistency across use cases.
