Agent Studio

AIP Agent Studio allows users to build interactive assistants — known as AIP Agents — that are equipped with enterprise-specific information and tools, deployable internally in the platform and externally through the Ontology SDK and platform APIs.

Agents are powered by large language models, the Ontology, documents, and custom tools, and can be integrated into applications to facilitate dynamic, context-aware read and write workflows.

Built on Palantir's rigorous security model, agents are granted access only to what is necessary to complete a task.

Read More

Pilot

Pilot is an AI-powered application builder that lets you create full-stack applications on top of your ontology using natural language prompts.

Rather than separately defining object types, writing action types, designing a UI, and wiring OSDK hooks, Pilot handles the full development lifecycle — from description to deployable application — so you can focus on what you want to build rather than how to build it.

When your application is ready, Pilot provides a guided deployment workflow that publishes it as a production-hosted application.

Read More

Time Series

Time series data are a series of measurements taken over time, usually at a regular interval — such as total flights per day, production outputs per hour, or high-frequency temperature readings at sub-second resolution.

To use data for time series analysis in Foundry, you set up two components: a time series object type, which defines your dataset's metadata and allows Foundry applications to access the data, and a time series sync, which indexes the data and provides values for time series properties. Once configured, time series data can be visualised and analysed across Foundry applications including Quiver, Vertex, and Workshop.

AIP Analyst

AIP Analyst is an interface for agentic workflows that lets you use natural language to perform ad-hoc analyses across your ontology.

You can ask AIP Analyst a question, and the agent will answer by autonomously searching your ontology, creating object sets, and transforming data before generating summaries and visualizations.

It supports a range of analytical capabilities including geospatial filtering, SQL queries, ontology aggregations, and interactive Vega chart visualisations.
