Palantir 与 Grafana Labs 的战略合作

引言

在当今快速演进的技术浪潮中，政府机构面临着管理日益复杂的 IT 基础设施的严峻挑战。高效的软件可观测性与监控能力，是保障运行效率与安全的关键所在。洞察到这一需求，Palantir 与 Grafana Labs 通过《FedStart 计划》结成战略合作伙伴，旨在全面革新联邦部门的技术能力。此次合作将 Grafana Labs 领先的可观测性平台与 Palantir 在联邦业务中积淀的深厚经验相结合，为政府机构提供更强大的数据可视化、监控与可观测性工具。

Palantir 与 Grafana Labs 的合作实现了优势互补，不仅降低了成本、提升了运营效率，还加强了对联邦机构的支持力度。此次合作凸显了将前沿可观测性解决方案引入联邦市场的战略价值。通过强强联手，Palantir 与 Grafana Labs 将交付创新的可观测性解决方案，赋能政府机构更好地理解并管理其复杂的系统与基础设施。

Grafana Labs 的技术：契合政府需求

在 Palantir，我们深知 Grafana Labs 的技术在可观测性与监控领域独树一帜，且我们内部也自豪地使用着该公司的工具。2023 年，Palantir 切换至 Grafana Labs 的可观测性工具，以更好地支撑我们在高度受监管环境下的工作，整合监控系统，并满足业务与运营快速扩张带来的持续需求。

“Grafana Labs 的解决方案帮助我们统一了跨所有环境的可观测性工具，大幅降低了成本并提升了运营效率，”Palantir 生产基础设施负责人 Greg DeArment 指出，“在亲身体验了 Grafana Labs 可观测性平台的强大能力后，我们非常期待将这项技术引入政府部门。Grafana Labs 的 LGTM 技术栈——包含用于日志的 Loki、用于可视化的 Grafana、用于链路追踪的 Tempo 以及用于指标监控的 Mimir——为应用和基础设施性能提供了实时洞察。”

尽管 LGTM 技术栈功能强大且广受赞誉，但 Palantir 在政府项目（尤其是国防领域）的切身实践经验表明，政府部门对这些工具的需求已迫在眉睫。在难以触及且物理隔离的环境中，一流的可观测性与监控工具对于维持软件的运行可用性至关重要，然而在这些环境中进行采购与部署同样面临巨大挑战。

Palantir 在实施政府项目方面积累的丰富经验，精准定位了联邦 IT 基础设施中的缺口与痛点。显而易见，Grafana Labs 的可观测性工具对于管理和优化这些复杂系统而言不仅是锦上添花，更是必不可少。将 LGTM 技术栈引入政府部门，确保了政府机构能够使用业界顶级的可视化与可观测性工具。

“通过将 Palantir 在联邦领域积累的深厚经验与 Grafana Labs 领先的可观测性平台相结合，我们打造了一款强大的解决方案，直击政府机构及其商业合作伙伴面临的独特挑战。此次合作标志着我们在让高度受监管环境下的组织能够用上企业级可观测性方面迈出了重要一步。” —— Grafana Labs 企业发展副总裁 Ash Mazhari

Palantir FedStart + Grafana Labs

“Grafana Labs 选择 Palantir 作为进入联邦市场的合作伙伴，对此我们倍感振奋，”Palantir 高级架构师、《Palantir FedStart 计划》负责人 Ali Monfre 表示，“成为 Palantir FedStart 合作伙伴后，Grafana Labs 将能够在其安全且已获得认证的环境中运行产品，从而快速为其软件即服务（SaaS）解决方案完成认证——这是向美国政府机构销售 SaaS 的先决条件。此举将大幅缩短该技术落地美国政府的时间线。我们很自豪能赋能美国技术底座以支撑关键的政府需求，并迫不及待地希望我们的联邦合作伙伴也能用上 Grafana Labs 的技术。”

FedStart 借助 Palantir 的 Apollo 和 Rubix 平台，简化了认证流程并确保合作伙伴技术的无缝集成。Apollo 提供持续交付与应用管理能力，而 Rubix 则提供了一个经过加固的 Kubernetes 平台，支持跨多种形态要素的关键任务程序。这些能力使 FedStart 合作伙伴能够在高度受监管的环境中快速部署和管理其解决方案。

此次合作不仅会让美国政府机构用上 Grafana Labs 的技术，还将惠及所有现有的 FedStart 合作伙伴，以及其他需要符合 FedRAMP 或影响等级（Impact Level）合规要求的可观测性与监控解决方案的商业实体。该举措确保 FedStart 计划内的每一个合作伙伴都能用上 Grafana Labs 可观测性工具的进阶能力，从而增强其监控、可视化及优化 IT 基础设施的能力。

通过将 Grafana Labs 强大的技术栈与 Palantir 的技术相融合，我们正在构建一个强大的生态系统，赋能 FedStart 及其他商业合作伙伴实现更高的运营效率、成本节约与系统韧性。此次合作不仅强化了单个 FedStart 合作伙伴的能力，更提升了联邦 IT 整体举措的有效性与安全。

产品协同

支撑 FedStart 计划的同一套产品与基础设施——Apollo 和 Rubix——同样可部署于其他众多环境，尤其是物理隔离、涉密和边缘环境。这种灵活性使得 Grafana Labs 的软件即便在非 SaaS 环境下也能实现无缝集成，在任何需要的地方提供稳健的、可自管理的可观测性工具。该能力同时适用于政府与商业软件，确保跨各类场景的全面监控与管理。

此次合作促成了 Palantir 与 Grafana Labs 产品的无缝集成，确保联邦机构能够使用顶级的工具来可视化、存储和查询指标、日志和链路追踪。这套统一的可观测性与监控解决方案可跨多种环境生效，涵盖 SaaS、本地、边缘及涉密环境。

成功的联合部署已然印证了此次合作的有效性，带来了运营效率的提升与成本的节约。我们的协作赋能联邦机构，利用增强的可视化与可观测性工具更好地管理其 IT 基础设施。

展望未来：对创新的承诺

Palantir 与 Grafana Labs 致力于深化合作，聚焦长期协作，为政府部门引入创新的可观测性工具。此次合作不仅直击联邦机构的当务之急，也为未来的政府技术进步奠定了基础。两家公司将持续改进其产品，确保联邦机构能够用上最佳的 IT 基础设施管理工具。

Palantir 与 Grafana Labs 的合作树立了典范，展示了私营企业如何携手推动公共部门的创新。通过共享专长与资源，我们交付的解决方案实现了 1+1>2 的效果。这种协作精神正是我们使命的核心所在：支持政府机构追求卓越，并赋能美国工业基础。我们诚邀联邦机构及其他商业合作伙伴加入我们。携手并进，我们定能开创一个政府运营更加透明、高效、卓有成效的未来。
