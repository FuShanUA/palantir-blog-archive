Palantir Announces Availability of Foundry on Microsoft Azure

2023-01-25 · Original →

编者按：如需了解更多信息，请访问我们的 《Palantir 与微软合作专页》，或访问 《Microsoft Azure 市场》。

在全球经济充满不确定性的背景下，获取整合、受保护且受信任的数据与分析对于创造业务价值而言，比以往任何时候都更加关键。为了进一步推动变革性成果，Palantir 非常高兴能与微软合作，在 Microsoft Azure 上提供 Palantir Foundry，赋能现有及新客户在运营决策中更有效地应用数据与分析。

通过此次全新合作，各组织将能够快速部署 Palantir Foundry —— 我们由本体驱动的现代企业操作系统 —— 同时借助微软云级分析与 AI 解决方案，在 Azure 数据服务中释放更多价值。

作为此次合作的一部分，我们的 Foundry 平台已在 Azure 上线，客户能够快速部署我们的软件，同时受益于 Azure 值得信赖的安全基础设施及其全球商业版图。

在《Azure 市场》上发售将实现无缝采购与开票，客户可利用现有的 Microsoft Azure 预留额度 (MACC) 购买 Foundry 许可证并支付基础设施费用。

Foundry 的单视图本体可叠加在 Azure 数据服务之上，客户借此可利用已有投资加速实现价值，通过更好地释放洞察力、预测和模拟结果，做出更受数据驱动的决策。

该平台还将与 Microsoft Azure 上用于企业数据管理的原生 Azure 数据服务进行集成，例如 Azure Data Lake、Azure Synapse Analytics、Microsoft Power BI、Microsoft Dynamics 365、Microsoft Teams 和 Microsoft Industry Clouds。这意味着客户能够通过 Palantir 的软件定义数据集成 (SDDI)，在 Azure Synapse Analytics、Azure Data Lake Storage、Azure AI 和 Azure Machine Learning 等产品之上，进一步利用其在 Azure 数据服务中现有的 IT 投资。

“我们非常高兴能与 Palantir 合作，将 Foundry 引入 Microsoft Azure。借助由 Azure 云级分析和综合 AI 服务提供支持的 Palantir 数据驱动运营与决策平台，全球各地的组织将能够让自身数据发挥更大的效用。”
— 微软北美区总裁 Deb Cupp

Palantir Foundry 与 Azure 数据服务强强联手

我们与微软的全新合作关系还意味着，双方将在能源与可再生能源、零售与消费品等跨行业的联合商机，以及其他跨行业的可持续发展和 ESG 领域共同走向市场。在这些领域，微软客户可以借助 Azure 数据服务深化其现有的数字化转型成果：

  • 能源与可再生能源：Foundry 使客户能够快速、大规模地整合来自远程传感器和 Azure IoT Hub 的数据，并应用这些数据提升资产效率，业务范围涵盖从海上石油到陆上风电。
  • 零售与消费品：该平台让组织能够近乎实时地洞察需求，并具备实时调整促销、库存和运营的能力。
  • 可持续发展与 ESG：我们正通过创建通用的碳本体，帮助各组织向净零排放转型，赋能一线决策人员调整工作以达成减排目标。
  • 医疗与生命科学：Foundry 的应用贯穿医疗与生命科学价值链，从药物发现与研发，一直到制造、营销和销售。通过与 Azure Health Data Services 集成来管理受保护的健康信息。

我们还携手合作，利用 Palantir HyperAuto 整合 SAP 及其他 ERP 系统，帮助上述行业及更多领域的客户加速实现价值，助力其构建更加整合的数据格局。Palantir HyperAuto 能够帮助客户加速向 SAP on Azure 迁移的进程，并在短短数小时内快速呈现洞察。

合作实践

通过 Azure 可快速部署的 Palantir Foundry 附加功能，也来自 Wejo 等客户。Wejo 是 Palantir 引以为傲的合作伙伴，致力于优化 Foundry 的各项功能，同时也是智能出行有益云及软件解决方案领域的全球领导者，专注于网联、电动和自动驾驶车辆数据。

其数据源自超过 920 亿次车辆行程，包含超过 19.5 万亿个数据点，为各行各业的商业与组织提供创新、推动增长、变革社区以及拯救生命的力量。

“我们希望通过智能出行的有益产品和服务，减少每年因道路事故造成的 130 万人死亡，以及因排放导致的额外 800 万人死亡。作为 Foundry 平台的一部分，我们非常兴奋，使用 Azure 的 Palantir 客户将能够更快速地从 Wejo 获取整合、受保护且受信任的数据与分析，用于智能出行项目并创造业务价值。”
— Wejo 战略与创新执行副总裁 Sarah Larner

我们期待与微软合作，进一步扩大 Foundry 的可用范围，赋能跨行业客户更好地利用其现有投资，以改善运营成果。
