Launching ERP Suite to Deliver Cost Savings for AWS Customers

2021-03-05 · Original →

各行各业的企业都在企业资源计划（ERP）系统上投入巨资，但它们共同面临一个难题：如何从分散的 ERP 数据中挖掘全部价值。为了解决这一问题，Palantir 推出了 ERP 套件。这是一套经过优化的、可在 Amazon Web Services（AWS）上运行的交钥匙解决方案，能够实现 ERP 体系内数据的快速集成与分析。目前，ERP 套件已向所有 AWS 客户开放，并已被包括 bp 和 LANXESS 在内的多家企业采用，在数周内实现了显著的成本节约。

ERP 套件

多年来，Palantir 一直致力于帮助客户从其 ERP 资产中获取更多价值。通过聚合来自任何平台（包括现有数据湖）的数据，工程师和非技术用户都能够利用 ERP 数据的全部粒度来提出并解答业务问题，进而执行数据驱动的运营操作，而无需依赖企业内部的 ERP 主题专家或外部顾问。

ERP 套件开箱即用，自带一个专注于行业用例的组件库（如供应链优化），可在几天内部署完毕。Palantir 使用专为 ERP 应用开发的 Palantir Foundry 数据连接器来改善数据发现，缩短 ERP 数据的见效周期，实现工作流创建的平民化，并推动更快的业务成果产出。

针对 AWS 客户优化以实现快速部署

ERP 套件已经过优化，能够在 AWS 上运行。因此，现有的 AWS 客户可以快速、动态地利用 ERP 套件，以满足其独特需求。

“过去一年里，我们与 AWS 在多个垂直市场展开了合作。借助这款全新的 ERP 套件，我们在短短几周内就看到了直接交付给客户的惊人节约。我们期待与 AWS 携手合作，以更快的速度将此项成果扩展至更多客户。”

– Josh Harris，Palantir 副总裁

“我们很高兴能与 Palantir 合作，在 AWS 上提供这套工具，帮助我们的客户更快地获取洞察，解锁他们已有的数据。借助 AWS，Palantir 的 ERP 套件能够帮助客户摆脱复杂性，更快地从数据中获取智能，并做出关键决策以节约时间和成本。”

– Teresa Carlson，AWS 全球公共部门与行业副总裁

ERP 套件实践：bp 与 LANXESS

bp 的 ERP 套件应用

跨国能源公司 bp 是 ERP 套件的早期采用者。在安装 ERP 套件的两周内，bp 成功释放了约 5000 万美元的营运资金；若将其应用于全球供应链，还有望进一步释放更多成本节约。

LANXESS 的 ERP 套件应用

全球化工公司 LANXESS 也采用了 Palantir 的 ERP 套件，以期从其 ERP 数据中获取更多价值。在安装 ERP 套件的几天内，LANXESS 便掌握了跨多个 ERP 系统的端到端价值链可见性，并具备了模拟决策和运行复杂假设情景的能力。如今，LANXESS 能够评估并应对需求变化，主动做出数据驱动的调整决策，从而实时优化库存水平，避免代价高昂的库存积压或短缺。
