Net Zero is a Data Integration Problem: How Palantir enables data-driven decarbonization

2021-03-25 · Original →

2020年，各行各业的组织机构纷纷制定或强化了净零排放目标。制造商计划在复杂的供应链中尽量减少不可再生材料的使用；金融服务企业致力于使其投资组合与可持续投资实践相契合；政府则拨出预算，推动疫情后的绿色复苏。

长达数十年的净零排放目标固然值得赞赏，但必须有可靠的脱碳路径作为支撑。2021年面临的挑战在于，如何将长期目标与日常运营相协调。脱碳需要根本性的变革。决策者需要对运营细节有基于数据的深入理解，才能推动相关改变，进而将净零排放变为现实。

碳排放计算器缺失细节

设定净零排放目标的典型步骤如下：

  • 估算组织的总排放量
  • 确定总体减排目标
  • 根据总体目标倒推，为每个年度和业务部门设定子目标

许多组织使用碳排放计算器来估算排放量。其逻辑很简单：将某项活动的数量乘以该活动的碳排放系数。如果员工在某一年累计飞行 50 万英里，而航空旅行的碳排放系数为每客英里 150 克，则总排放量为 75,000 千克二氧化碳当量。以一家全球制造商为例，它会将碳排放系数乘以供应、制造、分销和销售相关活动的数量，然后求和得出总量。

然而，与长达数十年的净零排放目标类似，碳排放计算器的精度较低。总量和估算值让人无法将排放量追溯至引发排放的具体流程。其结果是，气候足迹记录不完整，无法进行分解分析。

在制定详细的净零排放路径时，碳排放计算器得出的总排放数字并无太大帮助。为了实现净零排放，组织需要以细粒度的方式追踪排放情况，将其呈现给操作人员，并与日常决策挂钩。

举例来说：一位航运规划员接到任务，要为一位关注气候的客户规划一条低排放航线。然而，她无法看到自己的决策如何影响其管辖范围内航线的排放量。相反，她只能根据速度、成本和船舶利用率等现有信息进行假设。这不仅增加了开发稳健低碳产品的难度，还意味着她可能错失潜在的优化空间。

另一个例子是高管薪酬：多家组织已宣布，除了利润率和增长率等常规绩效指标外，还将把奖金与碳排放等可持续发展指标挂钩。这是一种强有力且重要的激励机制调整。但与财务指标不同的是，大多数企业领导者无法发现和评估其管辖范围内的排放情况。目前缺乏详细的核算和报告结构来帮助他们精准定位高排放源，并识别易于实现的改进机会。

净零排放本质上是一个数据集成问题

自成立以来，Palantir 一直致力于帮助组织规划复杂的业务转型路径。在应对净零排放目标时，我们的方法融合了过往的经验教训：

  • 宏观指标无法支持领导者做出细致入微的决策。
  • 静态的年度汇总数据不足以理解复杂的运营状况。
  • 微观指标，无论是碳排放量还是医院的个人防护装备库存水平，都难以收集和管理。
  • 转型需要将来自不同业务孤岛的相关信息提供给决策者（例如，员工差旅与压缩机的燃油消耗是完全不同的业务流程，但都与净零排放相关）。

将这些观察结果应用于净零排放路径：每一项消耗不可再生材料或燃料的活动都应被追踪，最好细化到单个操作、流程或生产项目的层面。随后，可以在细粒度层面进行碳排放计算，生成丰富的碳数据资产，为各类行动提供信息支持。这一碳数据资产使决策者能够将运营置于具体情境中——例如，将碳排放指标叠加在替代设备的材料成本和建设计划上，或者将其映射到商业预测的假设情景中。

将承诺转化为路径

我们的客户使用 Palantir Foundry 构建以碳为核心的通用运营视图，使组织的各个部分能够围绕净零排放目标进行协作。该过程包含三个关键要素：

1. 跨组织集成数据，包含开源与气候政策数据

数据集成使组织能够将孤岛数据源整合为排放量及其相关成本和风险的完整图景。利益相关者可以就碳排放影响提出精确问题，例如：“在满足成本和时间要求的同时，我可以在供应链中替换哪些中间产品以达成排放 KPI？”或“每个投资组合中哪类资产持有最集中的气候风险？”

利用开源气候数据和逻辑来增强专有信息，可以提升组织的决策能力。例如，制造商可以集成当前及拟议的碳税政策数据库，为特定厂区的排放管理计划提供依据。或者，物流提供商可以集成预定义的碳逻辑，将大规模运输指标转化为实时排放数据。

2. 模拟不同碳排放场景的建模能力

模型是现实世界的模拟版本，能够支持“假设”分析。将多个模型关联起来，可以简化脱碳决策背后的成本效益问题，从操作层面的“哪条航运路线能实现利润最大化且排放最小化？”，到战略层面的“我应该投资哪些资产以实现长期二氧化碳目标？”。随着时间的推移，Foundry 的模型管理功能允许用户分析和学习过往决策，从而为未来决策提供参考。

3. 快速评估变更影响并更新决策的灵活性

气候科学、政府政策和脱碳技术都在快速演进。Foundry 在建模和分析方面提供了高度的灵活性，使组织能够迅速评估新进展的影响，并相应地调整其策略。

要实现长期的净零排放目标，组织必须通过数据集成、细粒度的碳逻辑以及情景建模，将战略与运营紧密挂钩。碳排放指标只有与整个组织的运营数据相连，并追溯到具体的个人行动时，才能发挥其真正的效力。这些指标正是决策者构建净零排放路径的基石，使其能够根据自身的运营现实量身定制。
