Palantir Developers: Learn to build in Palantir Foundry

2022-02-10 · Original →

推出全新开发者资源，提升在 Foundry 中的影响力

组织中的每个人都应该能够利用正确的数据做出最佳决策。正因如此，Palantir 致力于让 Foundry 尽可能直观且易于使用——不仅面向数据科学家和工程师，也面向销售、产品开发、招聘等更多业务人员。

我们相信，加速实现这一愿景的最佳方式，是赋能每天在 Foundry 上构建应用的数千名开发者。为此，我们开始更多地参与 Stack Overflow 上的开发者社区互动，并推出一个专为开发者社区打造的新视频频道。

解答开发者疑问

多年来，我们一直通过 Foundry 内部的产品文档为客户提供开发支持，并在业务一线与他们进行实时协作。然而，随着 COVID 改变了我们的工作方式，开发者与 Palantir 团队成员需要寻找新的协作途径。最近，我们发现这些开发者在互联网上搜索有关 Foundry 的问题，无论是学习如何调试数据管道，还是寻求实施设计方面的灵感。久而久之，我们看到 Stack Overflow 已逐渐成为他们寻找答案的平台。

我们很高兴能将 Stack Overflow 上的开发者互动作为我们日常工作的重要组成部分。欢迎在此处分享开发者的想法、问题与协作。

Palantir 开发者视频频道

我们同样很高兴推出 Palantir 开发者频道，这标志着我们在整个客户群中的开发者支持迈入新阶段。如果说一图胜千言，那么视频就是一座知识宝库。该频道将作为一个枢纽，帮助开发者在 Foundry 中释放更多价值。

Palantir 开发者频道

从编写单元测试的教程，到 Foundry 参考项目的实操演练，Palantir 开发者频道将推出多个视频系列，重点介绍 Foundry 平台上不同的开发主题。

我们的客户开发者在使用 Foundry 时展现出了极强的创新能力，这些用法在六年前我们首次发布时根本无法想象。他们在塑造我们的产品方面发挥着关键作用。我们与开发者深度共创，构建了新功能甚至全新工具。我们携手打造了全新的图表和过滤器，利用 Workshop 简化了应用构建流程，并设计了 Quiver 以帮助开发者在几秒钟内叠加数百万个传感器数据点。我们乐观地相信，这个频道将为开发者提供一个协作空间，供大家学习、分享，并激发构建全新解决方案的灵感。
