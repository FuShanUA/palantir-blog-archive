Palantir and Google Cloud Announce Strategic Partnership

2022-06-16 · Original →

为商业客户提供更智能的数据驱动决策

编者注：阅读 Google Cloud 的博文并探索其页面上的相关内容。

随着世界日益复杂，商业成功取决于能否有效管理数据，并将分析结果转化为更优的决策与成果。为了助力客户实现数据驱动转型，Palantir 与 Google Cloud 携手合作，为商业客户提供端到端的数据解决方案，将 Google Cloud 在分析、人工智能和机器学习方面的能力注入 Palantir Foundry 的运营决策中，为其提供强大助推。我们与 Google Cloud 的合作，在技术专家、IT 部门和一线业务用户之间搭建了一座快速实现价值的桥梁，赋能各方以更具意义的方式利用数据。

我们将从三个关键领域展开合作：

  • Palantir 在 Google Cloud 上交付其用于企业数据管理的 Foundry 平台。客户能够快速、便捷地部署 Foundry 并充分利用其功能，同时受益于 Google Cloud 全球化、可扩展、安全且可持续的基础设施。客户还可以直接从 Google Cloud Marketplace 部署 Foundry。
  • Foundry 将与 Google Cloud 产品集成，赋能组织全员更好地处理数据。Foundry 用户可以将 Google BigQuery 中的数据与多种其他来源的数据进行融合，并运行 Google 的分析及 AI/ML 功能，从而推动更高效的决策并加速创造商业价值。
  • Google Cloud 和 Palantir 将联合推向市场，帮助客户解决关键行业挑战，首批聚焦零售、供应链和金融服务行业。

打通数据与洞察，助力快速实现商业价值

通过使用与 Google Cloud 原生及行业解决方案相连接的预构建 Palantir Foundry 工作流，用户可在数周内构建完整的数据基础，以支持更智能的企业 AI。现有的 Google Cloud 客户可以利用由 SDDI 驱动的 HyperAuto，直接连接源系统，快速将数据导入 BigQuery。随后，他们可以利用 Foundry 的运营应用与决策框架，发挥 BigQuery 和 VertexAI 的强大能力。Palantir Foundry 开箱即用的工作流和规则解决方案能带来立竿见影的商业价值，并融合 Google 的 AI/ML 能力，在解决复杂商业问题时让我们的 Foundry 工作流变得更加智能。

通过消除数据集成后端的技术复杂性，我们赋能一线用户发挥自身专长，以更自然的方式与数据进行交互，并在产出最佳成果时实时激活数据。一旦出现问题，在 Google Cloud 上运行 Foundry 可使组织进行动态协作——同样是实时进行——通过打破阻碍深入分析的组织孤岛，确保组织始终掌控业务全局。

解决关键行业挑战

在这个由数据定义的新时代，组织若想保持竞争力，就必须真正了解终端客户。我们不仅对这一全新合作能够加深对终端用户的认知感到振奋，也同样期待它能够帮助客户更好地认识自我，明晰自身在更广阔的行业格局中所处的位置。以下是一些案例：

  • 零售商 可以实施 Palantir 的库存管理工作流，并结合 Google Cloud 的 Recommendations AI、Forecast AI 和 Demand AI，更好地预测热门商品何时断货，并灵活应对供应链中的各种中断。
  • 银行 可以将交易数据、线上活动数据和欺诈警报数据汇聚在 Foundry 中，从而更好地检测欺诈和洗钱活动；同时，将数据导入 Google BigQuery 并在同一数据上叠加 AI/ML 模型，以获取深度洞察。
  • 生鲜杂货商 可以利用在 Google Cloud 上运行的 Palantir Foundry，整合销售、库存、促销和产品数据，更好地解决运营损耗这一长期痛点，从而助力减少食物浪费。

通过针对常见商业挑战构建应用场景，Palantir 和 Google Cloud 能够加速客户的数字化转型，并为其制定关键任务决策奠定所需的基础。

合作共赢

组织转型的基础在于能否将数据有效应用于运营决策。此次合作将使企业能够在 Google Cloud 安全可信的基础设施上，充分利用 Foundry 在数据驱动决策和态势智能方面的能力。

有兴趣进一步了解 Palantir 与 Google Cloud 联合解决方案的客户，请访问 palantir.com/google-cloud 和 cloud.google.com/solutions/palantir。
